Struttura Logica di Internet

Comprendi la struttura logica di Internet e i suoi protocolli.

Il Protocollo di Rete

I protocolli di rete definiscono le regole per la trasmissione dei dati. I protocolli sono suddivisi in livelli, secondo il modello OSI.

Il Protocollo TCP/IP

TCP/IP è il protocollo principale di Internet, composto da:

  • TCP: Gestisce la trasmissione dei dati, garantendo l'ordine e l'assenza di errori.

    • Clicca qui per mostrare il contenuto

    TCP (Transmission Control Protocol)

    Cosa fa: TCP è come un postino che consegna pacchi (dati) da un computer all'altro.

    Come lo fa: TCP si assicura che i pacchi arrivino nell'ordine giusto e senza errori. Se un pacco si perde, lo recupera. Esempio: Immagina di inviare una serie di lettere a un amico. TCP fa in modo che tutte le lettere arrivino nell'ordine giusto e senza nessuna mancante.

  • IP: Si occupa dell'instradamento dei pacchetti di dati.

I Protocolli HTTP e HTTPS:

  • HTTP è il protocollo per trasferire le pagine web.

    • Clicca qui per mostrare il contenuto

    HTTP (HyperText Transfer Protocol)

    Cosa fa: HTTP è come un cameriere che porta le pagine web dal server al tuo browser. Come lo fa: Quando digiti un indirizzo web, HTTP chiede al server di inviarti quella pagina web. Esempio: Immagina di andare al ristorante e chiedere un piatto. HTTP è il cameriere che prende il piatto dalla cucina e te lo porta al tavolo.

    Differenza principale

    TCP: Gestisce come i dati (lettere) viaggiano da un computer all'altro in modo ordinato e sicuro. HTTP: Usa TCP per chiedere e portare le pagine web (piatti) dal server al tuo browser.

  • HTTPS è una versione sicura di HTTP.

World Wide Web

Il WWW è un sistema di documenti ipertestuali accessibili tramite Internet.

L'URL

L’URL (Uniform Resource Locator) è l’indirizzo utilizzato per identificare una risorsa su Internet. È composto da protocollo, nome del dominio e percorso della risorsa.

DNS

Il DNS traduce i nomi di dominio in indirizzi IP.

HTML

HTML è il linguaggio per creare pagine web.

CSS

CSS è il linguaggio che definisce lo stile e il layout delle pagine web.
